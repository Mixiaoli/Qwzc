# 🚀 企微智齿工单采集工具（Mixiao）

一个用于**快速从企业微信复制聊天内容并自动生成工单 Excel** 的效率工具。
模板需下载 智齿 模板后在打包后软件同一文件夹下使用

---

# 📌 功能介绍

本工具解决的问题：

👉 客服在企业微信中处理问题
👉 需要手动整理成工单导入系统
👉 操作重复、效率低、容易出错

---

## ✅ 本工具实现

* ✔ 快捷键一键采集数据（F2）
* ✔ 自动复制当前选中内容（无需手动 Ctrl+C）
* ✔ 自动区分：

  * 第一次：用户信息
  * 第二次：聊天内容
* ✔ 自动生成标准工单字段
* ✔ 支持批量生成多条工单
* ✔ 一键导出 Excel（完全符合导入模板）

---

# 🧠 使用流程

```text
1️⃣ 打开工具
2️⃣ 输入【受理客服邮箱】
3️⃣ 点击【启动采集】

4️⃣ 在企业微信中：
    👉 选中用户名 → 按 F2
    👉 选中聊天内容 → 再按 F2

5️⃣ 点击【导出Excel】
6️⃣ 上传到工单系统
```

---

# ⌨️ 快捷键说明

| 操作  | 说明          |
| --- | ----------- |
| F2  | 采集当前选中内容    |
| ESC | 退出程序（控制台模式） |

---

# 📂 生成字段说明

工具会自动生成以下字段（符合导入模板）：

```text
工单标题（必填）
工单描述（必填）
客户昵称（必填）
客户邮箱
客户手机
状态（已解决）
优先级（低）
受理技能组
受理客服（你填写的邮箱）
反馈渠道（企业微信）
请求类型（服务请求）
解决方案
挂单状态（关闭）
受理角色（分校IT解决）
三线处理
网络组意见
```
